Module 01 · 6–7 minutes

Introduction to AidData's
GCDF Dataset

Understand the purpose, scope, and methodology of the Global Chinese Development Finance dataset. Version 3.0 covers 2000–2021 and utilizes the TUFF methodology to triangulate data from official announcements, media reports, and satellite imagery.

Learning Objectives
ODA vs. OOF: Understand the distinction between Official Development Assistance (grants and concessional loans) and Other Official Flows (commercial or market-rate loans).
TUFF Methodology: Learn how Tracking Underreported Financial Flows triangulates data from multiple sources when official government data is limited.
02
Module 02 · 7–8 minutes

Navigating the
Dashboard

Master the dashboard interface components and understand data visualization options. Learn to navigate seamlessly between map, table, and chart views to explore Chinese development finance data effectively.

Learning Objectives
Three Visualization Options: Explore Map View (project locations with sector-based color coding), Table View (expandable project records), and Chart View (statistical analysis and trends).
Saving Configurations: Learn to copy URLs to save your current filter settings and dashboard configuration for later access or sharing with colleagues.
03
Module 03 · 8–9 minutes

Searching and
Filtering Data

Construct effective keyword searches and apply single and multiple filters. Learn to combine search strategies for precise results and understand filter logic to extract exactly the data you need.

Learning Objectives
Keyword Search Strategies: Use related terms to expand results—for infrastructure projects, try 'road', 'highway', 'expressway', or 'motorway' to capture all relevant projects.
Combining Filters: Apply filters simultaneously with keyword searches—select geographic regions first to narrow country options, then add project status and confidence level filters.
Confidence Levels: Understand High (verified through multiple authoritative sources), Medium (supported by credible evidence), and Low (limited or unverified sources).
04
Module 04 · 5–6 minutes

Exporting Your
Results

Export filtered data in appropriate formats and understand field definitions in exported files. Learn best practices for data citation and use to maintain research integrity.

Learning Objectives
Export Formats: Choose CSV format for maximum compatibility with data analysis tools, or Excel format for formatted sharing with non-technical stakeholders. Exports include only projects matching your filters.
The Codebook: Access detailed definitions for all variables through the codebook in the methodology section—your essential reference for field meanings, coding schemes, and data structure.
Proper Citation: Include dataset name, version number, access date, and URL. Example: "Global Chinese Development Finance Dataset, Version 3.0, accessed January 13, 2026"
